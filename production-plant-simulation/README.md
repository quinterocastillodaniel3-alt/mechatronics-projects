# Production Plant Simulation

This project contains 3D models and simulation components developed for the Mechatronics Systems Simulation course.

## Project Description
The objective of this project was to design and simulate a production plant using mechatronic system principles, automated processes, and industrial workflow analysis.

## Technologies and Software
- Inventor AUTODESK
- STL 3D Models
- Industrial Simulation Concepts

## Contents
- Conveyor system components
- Mechanical supports
- Production plant structures
- Simulation-related models
